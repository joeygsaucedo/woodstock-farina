# Images Folder

## Quick Reference - What Goes Where

### 📁 hero/
Main banner images (landscape, 1920x1080px or larger)
- `hero-main.jpg` - Homepage: oven with fire, pizza making action

### 📁 pizzas/
Pizza close-ups (square or portrait, 800x800px+)
- `margherita.jpg` - Traditional margherita
- `street-corn.jpg` - Mexican street corn pizza
- `al-pastor.jpg` - Al pastor pizza

### 📁 events/
Event photos from catering jobs (any orientation, 1200x800px+)
- `wedding-1.jpg` through `wedding-4.jpg` - Wedding events
- `corporate-1.jpg` through `corporate-4.jpg` - Corporate events
- `party-1.jpg` through `party-4.jpg` - Private parties

### 📁 team/
Behind-the-scenes and team photos (800x600px+)
- `victor.jpg` - Photo of Victor Melchor
- `oven-setup.jpg` - Setting up the oven
- `pizza-making.jpg` - Making/stretching pizza
- `oven-fire.jpg` - Close-up of the wood fire
- `serving.jpg` - Serving pizza to guests

## Download from Instagram

Visit: https://www.instagram.com/woodstock_farina/

Use https://igram.io/ to download:
1. Copy Instagram post URL
2. Paste into igram.io
3. Download image
4. Rename and place in appropriate folder
